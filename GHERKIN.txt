@cliente
Funcionalidade: Atualização de informações de cliente 

  Cenário: Atualização de informações de cliente no sistema
    Dado que o usuário está na página de atualização de informações
    Quando o usuário clica no botão "Atualizar dados " 
    E habilita o formulário ele preenche com as informações do cliente:
     | Nome       | Endereço | Bairro   | Cidade   | UF | CEP       | Telefone       |
     | Cliente A  | Rua A    | Eldorado | Contagem | MG | 00000-000 | (31)99999-9999 |
    Então o sistema deve exibir a mensagem "Cadastrado atualiza com sucesso"

@Usuário
Funcionalidade: Excluir cliente

  Cenário: Excluir cliente
   Dado que o usuário pesquise o id do cliente tera uma opção para excluir
   Quando o usuário clica no botão "Excluir" 
   E habilita a opção para selecionar o cliente desejado

     □	|Carlos Eduardo | kaduardoa@gmail.com | 123456 |
   E o usuário clica no botão "Excluir"
   Então o sistema deve exibir a mensagem "Cliente excluido com sucesso"

@Usuário
Funcionalidade: Alterar estoque de produtos

 Cenário: Alterar produto no sistema
    Dado que o usuário está na página de produtos
    Quando o usuário clica no botão "Alterar" 
    E habilita o formulário para preencher as informações novas do produto:
      | Nome       | Estoque | Valor | Localização |
      | Produto A  | 100     | 20    | corredor 5, pratileira 10 |
    E o usuário clica no botão "Salvar"
    Então o sistema deve exibir a mensagem "Produto alterado com sucesso"
    E o produto "Produto A" deve estar na com novas informações

@Usuário
Funcionalidade: Cadastrar Usuário 

Cenário: Cadastrar um usuário
    Dado que o usuário está na página de cadastro
    Quando o usuário clica no botão "Novo" 
    E habilita o formulário para preencher com as informações:
      | Nome       | Login | Senha | Endereço | Bairro   | Cidade   | UF | CEP       | Telefone       |
      | Carlos     | Cadu  | 9595  | Rua A    | Eldorado | Contagem | MG | 00000-000 | (31)99999-9999 |
    E o usuário clica no botão "Salvar"
    Então o sistema deve exibir a mensagem "Usuário cadastrado com sucesso"
    E o Usuário "Carlos" deve estar no banco de dados de cadastrados
